# Privacy and Confidentiality

## Showcase Scope

This repository is intentionally designed as a **portfolio showcase** of the Centinela FDA Recall Analysis project.

Its purpose is to communicate:

- the business problem
- the analytical methodology
- the system architecture
- example outputs
- the role of the LLM
- key project insights

## What Can Be Shared

The following elements are safe to present publicly in this showcase:

- public FDA recall data sources
- the analytical logic at a high level
- time-series methodology
- screenshots of sanitized outputs
- public sample data
- documentation of the system’s business value

## What Is Kept Private

To protect confidential intellectual property, the repository does **not** include:

- production JavaScript / Google Apps Script source code
- proprietary implementation details
- deployment configuration
- operational credentials or API keys
- internal production prompts in full operational form

## Why This Matters

Confidential project work can still be presented responsibly in a portfolio when the focus is placed on:

- problem framing
- analytical thinking
- architecture
- insights
- decision-support value

This repository follows that approach. It demonstrates technical capability without exposing implementation assets that should remain private.
